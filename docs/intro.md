# Physical AI & Humanoid Robotics

Welcome to the **Physical AI & Humanoid Robotics** interactive textbook! This comprehensive resource will guide you through the fundamentals of building intelligent robotic systems, from basic ROS 2 concepts to advanced vision-language-action integration.

## Course Overview

This textbook provides hands-on learning for robotics and AI integration through five major modules and a capstone project. Each module includes detailed explanations, runnable code examples, exercises, and practical deployment guidance.

## Learning Objectives

By completing this textbook, you will:

- **Master ROS 2 fundamentals**: Understand nodes, topics, services, and robot description formats
- **Develop simulation skills**: Create digital twins using Gazebo and Unity
- **Apply AI techniques**: Train robots using NVIDIA Isaac Sim and reinforcement learning
- **Integrate multimodal systems**: Combine vision, language, and action for intelligent behavior
- **Build complete systems**: Design and implement an autonomous humanoid robot

## Prerequisites

Before starting this course, you should have:

- **Programming**: Basic knowledge of at least one programming language (Python or C++ recommended)
- **Linux**: Familiarity with command-line operations (cd, ls, mkdir, etc.)
- **Mathematics**: Understanding of basic linear algebra and calculus concepts
- **Computer**: System capable of running ROS 2 and simulation environments (see [Software Setup](./resources/software-setup.md))

:::tip No Prior Robotics Experience Required
This textbook is designed for learners new to robotics. We'll introduce all concepts from the ground up with clear explanations and practical examples.
:::

## Course Structure

### [Module 1: ROS 2 (Robotic Nervous System)](./module-1-ros2/index.md)

Learn the Robot Operating System (ROS 2), the foundation for modern robotics software. Master nodes, topics, services, parameters, and robot description formats.

**Duration**: 2-3 weeks
**Topics**: Nodes, Topics, Services, Actions, Parameters, URDF

### [Module 2: Gazebo & Unity (Digital Twin)](./module-2-simulation/index.md)

Discover how to create realistic robot simulations using Gazebo and Unity. Learn sensor modeling, world building, and physics simulation for safe robot development.

**Duration**: 2-3 weeks
**Topics**: Gazebo Simulation, Unity Integration, Sensor Systems, Environment Design

### [Module 3: NVIDIA Isaac (AI-Robot Brain)](./module-3-isaac/index.md)

Explore NVIDIA Isaac Sim and Isaac Gym for training intelligent robots using reinforcement learning. Master sim-to-real transfer and domain randomization techniques.

**Duration**: 3-4 weeks
**Topics**: Isaac Sim, Reinforcement Learning, Training Scenarios, Sim-to-Real

### [Module 4: Vision-Language-Action (VLA)](./module-4-vla/index.md)

Integrate computer vision, natural language processing, and action planning to create robots that understand visual scenes, interpret commands, and execute complex tasks.

**Duration**: 2-3 weeks
**Topics**: Vision Processing, Language Models, Multimodal Fusion, Action Coordination

### [Capstone Project: Autonomous Humanoid](./capstone/index.md)

Apply everything you've learned to design and implement a complete autonomous humanoid robot system. This project integrates all module concepts into a functioning robot.

**Duration**: 4-6 weeks
**Deliverables**: System architecture, working implementation, evaluation report

## How to Use This Textbook

### For Self-Study Learners

1. **Follow the module sequence**: Start with Module 1 and progress sequentially
2. **Execute all code examples**: Copy and run code snippets in your local environment
3. **Complete exercises**: Practice concepts with hands-on exercises at the end of each chapter
4. **Use the glossary**: Reference technical terms in the [Glossary](./resources/glossary.md)
5. **Validate your work**: Check your understanding against provided solutions

### For Instructors

This textbook is designed for academic courses (15-week semester, 3 credit hours):

- **Week 1-4**: Module 1 (ROS 2)
- **Week 5-7**: Module 2 (Simulation)
- **Week 8-11**: Modules 3-4 (Isaac + VLA)
- **Week 12-15**: Capstone Project

See each module's index for teaching tips, common misconceptions, and assessment suggestions.
